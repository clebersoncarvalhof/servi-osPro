# ServiçosPro - Plataforma de Agendamentos

Este é um site moderno para agendamento de serviços profissionais, iniciando pela categoria de cabeleireiros.

## Como Rodar Localmente

### Pré-requisitos
- Node.js instalado (versão 18 ou superior)

### Passos
1. **Instale as dependências**:
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```

3. **Acesse no navegador**:
   Abra [http://localhost:3000](http://localhost:3000)

## Tecnologias Utilizadas
- **Next.js 15+** (App Router)
- **Tailwind CSS** (Estilização)
- **Framer Motion** (Animações)
- **Lucide React** (Ícones)
- **Prisma & SQLite** (Banco de dados - em desenvolvimento)

## Estrutura do Projeto
- `src/app`: Páginas e rotas.
- `src/components`: Componentes reutilizáveis (Navbar, Hero, etc).
- `src/lib`: Lógica do banco de dados e dados simulados.
